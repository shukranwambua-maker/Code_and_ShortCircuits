This is a fun hobby project to improve the "GENERIC" and mostly available RC Cars found in the market. If this works, I'd satisfy a childhood dream as well as enable maybe hundreds of other little boys enjoy RC toy casrs. It is made up of:
     -ESP32
     -L298N Motor Driver
     -SG90 servo motor 
     -7.4v 18Wh Li-ion Battery pack. 
     -6v Micro Dc motor
    The chasis has been somewhat modified to help integrate the new compnents.
Hopefully I'll improve this iteration in the future and try to make it economically viable for little boys.
